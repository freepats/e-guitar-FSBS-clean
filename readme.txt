Electric Guitar FSBS bridge pickups (clean)
-------------------------------------------

Version 2022-09-11

Sound bank made by direct sampling a Fender electric guitar (basic model)
using an audio interface with dedicated instrument inputs.

Samples had been processed through an amplifier and effects rack to create a
clean guitar sound. Direct (unprocessed) sounds and other variants are also
available. Please visit FreePats web pages for updates and more content:
http://freepats.zenvoid.org/
